The project was build with:

![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=for-the-badge&logo=vite&logoColor=white)


## Deployment

To install all dependencies

```bash
  npm install
```

To deploy this project run

```bash
  npm run dev
```
